                                                   Title 
                                      E-Wallet Console Application (C#)
                                              
Description : 

This project is a simple E-Wallet system built using C# Console Application
to practice Object-Oriented Programming (OOP) and Object-Oriented Design (OOD).


Features : 

• Create Wallet
• Deposit Money
• Withdraw Money
• Transfer Money
• Transaction History
• Change Password


Concepts Practiced :

• Encapsulation
• Abstraction
• Object Collaboration
• Basic OOD Design


Technologies :

 C#
.NET
Console Application
